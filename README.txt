JediSoftware Social Bar is a plugin that adds a fancy bar to your website, containing links to your social pages.

== Description ==

JediSoftware Social Bar is a plugin that adds a fancy sidebar to your website, containing links to your social pages.<br />
The bar is a sidebar which stays on the left of the screen and moves while you scroll the page.<br />
It currently supports 11 social networking sites:
<ul>
	<li>Facebook</li>
	<li>Pinterest</li>
	<li>Tumblr</li>
	<li>Twitter</li>
	<li>Google+</li>
	<li>Linkedin</li>
	<li>Youtube</li>
	<li>Instagram</li>
	<li>Soundcloud</li>
	<li>StumbleUpon</li>
	<li>VKontakte</li>
</ul>

You can choose which buttons to enable, set the url for each of them and change their order.<br />
You can also choose an icon set (you can choose between default, medium and small size icon set) and the speed you want your sidebar to move on scroll.

== Installation ==
<h3>Method 1:</h3>
To install the plugin, just download the zip file, then go to your administration panel, choose Plugins->Add New->Upload and upload it.<br />
Then use the "Activate" button to enable the plugin.<br /><br />
After activation, you'll see the "JS Social Bar" entry on the left-menu.

<h3>Method 2:</h3>
Another way to install the plugin is to decompress the the zip file and upload the <code>js_social_bar</code> folder to the <code>/wp_content/plugins</code> folder.<br />
Then you should see the new plugin under the Plugins section in your Wordpress installation: Actvate the plugin and enjoy your new bar!
